# Things 3 Interface Images

Reference images for Things 3 on desktop, tablet, and mobile.

Files:

- `desktop-mac.png` - Mac crop from the official collage.
- `tablet-ipad.png` - iPad crop from the official collage.
- `mobile-iphone.png` - iPhone crop from the official collage.
- `source-cultured-code-collage.png` - downloaded source image.

Sources:

- Cultured Code Things homepage collage:
  https://culturedcode.com/things/2024-01-20/images/whatsnew-collage-io60.png
- Cultured Code Things homepage:
  https://culturedcode.com/things/

Usage note: use these images as the reference UI set when modifying this app
toward the Things-inspired personal task-management direction. Study layout,
density, hierarchy, responsive/platform adaptation, and task-list affordances.
Do not copy the Things 3 UI directly.
